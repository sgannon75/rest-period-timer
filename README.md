# Rest Period Timer

A simple browser-based countdown timer for workout rest periods, built with plain HTML, CSS, and JavaScript — no dependencies or build step required.

## Features

- Quick presets for 60 seconds, 90 seconds, and 2 minutes
- Custom duration input (5–600 seconds)
- Visual progress bar that depletes as time counts down
- Audio and vibration alert when the rest period ends
- Optional auto-restart for continuous interval sets
- Tracks how many rest periods you've completed

## Usage

Open [`index.html`](index.html) directly in a browser — no installation needed.
